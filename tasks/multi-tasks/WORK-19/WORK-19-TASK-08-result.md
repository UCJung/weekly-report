## 4. 발견 이슈 및 수정 내역

### 이슈 #1 — 행선택·승인완료 색상 중복
**증상**: 행 선택 시 배경이 `var(--ok-bg)`(녹색)으로 변하는데, 승인완료 Badge도 동일한 `var(--ok-bg)` 배경색을 사용하여 Badge가 배경에 묻혀 보이지 않음
**원인**: TASK-06에서 승인완료 Badge에 `--ok-bg`/`--ok` 색상을 사용한 것이 행선택 배경색과 충돌
**수정**: 승인완료 Badge 색상을 `--primary-bg`/`--primary`(보라색)으로 변경하여 행선택 녹색 배경과 명확히 구분
