# WORK-19-TASK-09: 승인된 프로젝트의 월간승인 버튼을 승인완료 텍스트로 변경

> **Phase:** 기능수정
> **선행 TASK:** TASK-06, TASK-08
> **목표:** 이미 PM 승인된 프로젝트에서 월간승인 버튼 대신 "승인완료" 텍스트를 표시

## 요청사항
프로젝트 투입현황 > 인원별 투입현황에서 이미 승인된 프로젝트에도 월간 승인 버튼이 나타남.
승인된 프로젝트의 경우 버튼을 승인완료 텍스트로 변경해야 함.

---

## Step 1 — 계획서

### 1.1 현황 분석

- `MonthlyView` 컴포넌트: 항상 "월간 승인" 버튼을 표시
- `data.members[].pmApproval`: 승인 시 `{ status, approvedAt, autoApproved }` 객체 존재
- 전체 멤버에 대해 PM 승인이 완료되면 버튼을 비활성화해야 함

### 1.2 수정 방안

- `data.members` 중 하나라도 `pmApproval`이 존재하면 이미 승인된 것으로 판단 (PM 승인은 프로젝트 단위 일괄 승인)
- 승인 완료 시: 버튼 대신 보라색 "승인완료" 텍스트 + 체크 아이콘 표시
- 미승인 시: 기존 "월간 승인" 버튼 유지

### 1.3 산출물 목록

| 구분 | 산출물 |
|------|--------|
| 수정 | `packages/frontend/src/pages/ProjectAllocation.tsx` — MonthlyView 승인 버튼 조건부 렌더링 |
