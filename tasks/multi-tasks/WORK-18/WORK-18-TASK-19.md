# WORK-18-TASK-19: 근무시간표 종스크롤 복원 + 그리드 내부 툴바 제거

> 의존: WORK-18-TASK-18
> 상태: **대기**

---

## 1. 문제

1. **종스크롤 깨짐**: TASK-18에서 그리드를 카드(`maxHeight + overflow-hidden`)로 감쌌으나, 내부 `renderGridPanel`의 `height: 100%`가 카드의 maxHeight를 상속받지 못해 테이블 스크롤 불가
2. **그리드 내부 툴바 불필요**: 일반 모드에서 "N개 프로젝트" 텍스트 + 전체화면 버튼이 그리드 패널 내부 툴바에 남아있음 → 상단 카드 툴바로 이동 필요

---

## 2. 수정 내용

### 2.1 종스크롤 복원
- 그리드 카드에 `overflow-hidden` → `overflow: auto`로 변경하여 카드 자체가 스크롤 컨테이너가 됨
- 또는 내부 구조를 `display: flex; flex-direction: column; height: 100%`로 맞추어 `flex-1 min-h-0 overflow: auto`가 작동하도록 수정

### 2.2 그리드 내부 툴바 → 상단 카드 툴바로 이동
- 일반 모드(`!isFull`)에서 표시되던 "N개 프로젝트" + 전체화면 버튼 제거
- 상단 카드 툴바에 프로젝트 개수 + 전체화면 버튼 추가
- 전체화면 모드(`isFull`)의 툴바는 기존 유지

### 2.3 renderGridPanel 분리
- 일반 모드: 툴바 없이 테이블만 렌더링
- 전체화면 모드: 기존 툴바 + 테이블 유지
