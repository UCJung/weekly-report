# WORK-27-TASK-05: 이동/리사이즈 후 선택 방지 + 상세패널 순서/시간입력 개선

> **Phase:** 0
> **선행 TASK:** WORK-27-TASK-04
> **목표:** DnD/리사이즈 후 카드 선택 방지, 상세패널 예정일시→마감일시 순서 변경, 시간 입력을 시/분(0·30) 셀렉트로 전환

## 요청사항

- 카드를 이동하거나 리사이즈하고 나면 카드 선택 상태가 되서 우측에 카드 상세 레이어가 표시됨 - 카드 선택 상태가 되지 않도록 수정
- 우측의 카드 상세에 예정일과 마감일의 표시 위치 순서 변경 - 예정일시, 마감일시로
- 시간입력 방식 : 분에 1분단위로 선택 값이 보임 - 0과 30분 두개로 변경
